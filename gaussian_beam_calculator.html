Conventions used here

The input dFWHM is the intensity FWHM diameter at the waist. The corresponding waist parameter is

w0 = d_FWHM / sqrt(2 ln 2) I(y) ∝ exp[-2 (y / w)^2] d_FWHM(x) = sqrt(2 ln 2) · w(x)

This page displays the beam from the injection plane to the waist and the same distance beyond it. You can inspect any longitudinal slice, zoom beyond the default domain, and return to the waist with Fit view.

Plain-text calculation summary

Tip: if the box half-width is much smaller than the beam radius at the injection plane, the beam is strongly clipped before it ever reaches the waist. Reducing the waist diameter while keeping the waist far away usually makes clipping worse, not better, because the Rayleigh length becomes shorter and the beam diverges faster.
